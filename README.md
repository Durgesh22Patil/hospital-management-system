# Hospital Management System

## Overview
The Hospital Management System is a Java-based application developed to manage and digitize essential hospital workflows such as patient data management, appointment scheduling, and billing operations.  

The system connects Java application logic with an SQL database through JDBC, enabling reliable storage, retrieval, and manipulation of structured data. This project demonstrates practical backend development, database interaction, and modular software design principles.

---

## Features

- Patient registration and record management
- Appointment scheduling
- Billing and database storage
- Secure SQL connectivity via JDBC
- Structured modular Java code
